Simple CRUD operations for the REST API with data being stored in memory. Available routes:
  1. - A SHOW route: [GET] '/customers/{id}'
  2. - An INDEX route: [GET] '/customers'
  3. - A CREATE route: [POST] '/customers'
  4. - An UPDATE route: [PATCH] '/customers/{id}'
  5. - A DELETE route: [DELETE] '/customers/{id}'
